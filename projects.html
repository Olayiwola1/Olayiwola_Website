Projects




Land_Fix

Digitally immutable system to help deal with property corruption and promote transparency in Africa.


At NSBE Hacks UofT in Toronto, my team and I came up with an idea to help deal with the prominent problem of corruption around properties in Africa. We came up with a Blockchain-based app that creates an immutable digital record of the coordinates of lands/ properties and connects it to the rightful owner's name. We also leveraged the Google Maps API to create a visual representation of the owner's property. This system helps promote transparency since it is immutable, it can only be changed when the ownership of the property is being transferred to another person.

Technologies Used: Google Maps API, Python, JavaScript, HTML, CSS




CPU L8R

Created and developed a prototype for a completely functional pacemaker.



For an academic project in my Software Development class, we were tasked with building a fully functional pacemaker, and to implement its modes and functionalities. For the first phase of the project, we implemented pacemaker modes such as VOO, VVI, AAI, and VOO. We also built a graphical user interface using Tkinter and Python. For the second part of this project, we implemented serial communication using pySerial to enable our software to communicate with our hardware (FRDM-K64F board).

Gamification

Gamification system that was created with the aid of augmented reality technology.



SHacks was Scotiabank's first hackathon. My team and I were tasked with creating a gamification system that can be used to help attract more customers. We approached the problem by trying to leverage AR technology and use it to help us market their services. We decided to use the bank's logo as an image-marker, so whenever it is scanned by our app a trivia quiz would pop up, and customers would be able to play and learn more about the bank and their services.
